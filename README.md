## Built using üîß

### Frontend
- [React](https://react.dev/)
- [React Router](https://reactrouter.com/en/main)
- [MUI](https://mui.com/)
- [Axios](https://axios-http.com/)
- [Chart.js](https://www.chartjs.org/)

- [MongoDB Atlas](https://www.mongodb.com/products/platform/atlas-database)
